What is a virtual machine ?

Virtual machine is a software that stimulates the hardware of a physical computer. The purpose of virtual machine is to install and run multiple operating systems at the same time on a PC. There are multiple software that can run virtual machines. Example of virtual are Windows Virtual PC by Microsoft, Oracle Virtual Box, VMware. Each VM running on a computer works like a physical computer and is assigned virtual devices such as a virtual motherboard and virtual hard drive.

What are the benefits of using virtual machine?

Virtual Machine is a very useful software for users because of its ability to run multiple operating systems. Few benefits of virtualization are following

1) To try new operating system without buying a software

2) To backup your computer

3) To Run older software
